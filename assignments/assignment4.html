Challenge 5: Accessibility Concerns

Doing this assignment I learned of the many accessibility challenges with creating a website. Using something like a table in a website presents a serious accessibility concern as a screen reader cannot read a table clearly to a user. I did read about how the different form inputs allow for a higher degree of acccessibility though which I was glad to hear about. Using the lable tag allows people to click on the lable to interact with the form. As for the problems using a table in a website, I don't know exactly how a screen reader works, but if the structure of the table is what causes the problem then replacing the table with an alternative seems to be the only option.
